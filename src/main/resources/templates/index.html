CSD Laboratory for Zimbabwe

CSD Lab offers a conducive platform for implementing and testing a number of CSD use cases prior to exporting successful functionality to live systems or production environments. The environment offered here does not only consolidate all use cases in one platform but avoid destabilization of live systems as Integration Platform uses cases will be introduced and implemented.

TrainSmart Integration

TrainSmart allows users to accurately track data about training programs, trainers, and trainees, to better evaluate programs and report activities to stakeholders. Integration facilitates complete accurate provider data from the ZHRIS and detailed training data can be supplied to ZHRIS system. Thus better monitor health care worker training, credentials, competencies, and deployment patterns.
